# ターゲットユーザー

## メインペルソナ：旅行好きパパママ

### 基本情報
- **属性**: 20代〜40代、既婚、様々な年齢の子供を持つパパママ。
- **居住地**: 都市部近郊。
- **ITリテラシー**: Instagramで情報収集し、Google Mapを使いこなす。Webサービスの利用に抵抗はない。

### 旅行スタイル
- **頻度**: 年に2〜3回（大型連休、夏休み）。国内外問わず行く。
- **重視点**: 子供の体験（学び、遊び）、親のリフレッシュ、美味しい食事。
- **計画スタイル**: 
    - 事前リサーチは入念に行う（失敗したくない）。
    - でも現地では子供の体調や気分に合わせて柔軟に動きたい。

### 課題・ペイン
- 「インスタで見た場所に行ったら、実際は子供連れには厳しかった」という経験がある。
- 計画に時間をかけすぎて疲れてしまう。
- 旅行中の写真がスマホに入ったままで、見返す機会がない。

### このプロダクトを使う瞬間
- 旅行の1〜2ヶ月前、行き先が決まった後の詳細計画時。
- 旅行直前、持ち物や回るルートの最終確認時。
- 旅行後1週間以内、余韻に浸りながら思い出を整理したい時。

## サブペルソナ：情報収集メイン層
- **属性**: 自分で計画を組むのが苦手、または時間がない忙しい親。
- **行動**: 自分でゼロから作らず、ランキング上位の「失敗しないプラン」をそのままコピーして使う。
- **ニーズ**: 「これ通りに行けば間違いない」という安心感。
