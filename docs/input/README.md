# FamilyTrip Planner（仮）

## 概要
幅広い年齢の子供連れ旅行に対応した、旅程作成・共有プラットフォーム。
成功体験だけでなく「失敗談・注意点」も共有し、作成したプランをGoogle Mapで持ち歩ける。
投稿者には「思い出ムービー」が自動生成されるインセンティブを提供し、質の高い情報を循環させる。

## 解決する課題
- **計画の負担**: 子連れ旅行特有の考慮事項（トイレ、食事、休憩）が多く、情報収集が大変。
- **情報の非対称性**: SNSのキラキラした情報だけでは、現地のリアルな大変さや失敗リスクが見えない。
- **情報の分散**: ブログ、Instagram、Google Mapを行き来して管理するのが面倒。

## ターゲットユーザー
- 様々な年齢の子供を持つ旅行好きパパママ。
- 国内外問わず旅行に行きたいが、失敗したくない慎重派〜効率重視派。

## 主要機能
- **旅程コピー＆カスタマイズ**: 他の家族の成功プランをコピーして自分用に調整。
- **失敗アラート**: 「ここはベビーカーNG」「料理遅い」などの注意点をアイコン表示。
- **Google Mapエクスポート**: 作成したプランをGoogle Mapのお気に入りリストとして持ち出せる。
- **思い出ムービー生成**: 旅程と写真・感想を投稿すると、自動で素敵な振り返りムービーを作成。
