# プロダクト要件

## 1. プロダクト概要
### プロダクト形態
- **Webアプリ（レスポンシブ対応）**
    - 計画・編集はPC/タブレットでの操作性を重視。
    - 現地での閲覧・投稿はスマホでの操作性を重視。

### 対応プラットフォーム
- Webブラウザ（Chrome, Safari等）
- Google Mapアプリ（エクスポート先として連携）

## 2. コアバリュー
### コンセプト
「先人の知恵（成功と失敗）を借りて、家族旅行をアップデートする」

### Aha Moment
- 他人の詳細な旅程をワンクリックで自分のプランとして取り込めた瞬間。
- 旅行後、入力したデータが一瞬でエモい動画になって再生された瞬間。

## 3. 主要機能
- **旅程検索・コピー**: 年齢・場所・テーマで検索し、他ユーザーのプランを複製。
- **プランニング**: ドラッグ＆ドロップでスポットを入れ替え、時間を調整。
- **失敗アラート**: スポットに対するネガティブ情報（注意点）のアイコン表示。
- **Google Map連携**: 作成したルート・スポットをGoogle Mapのリスト/マイマップとしてエクスポート。
- **思い出ムービー生成**: 旅程データ＋写真＋一言コメントから動画を自動生成。

## 4. 非機能要件
- **デザイン性**: 「空港」「パスポート」をモチーフにしたワクワクするUI。
- **レスポンス**: 地図操作やプラン編集がサクサク動くこと。
- **データ保持**: ユーザーが入力した思い出データ（写真・テキスト）を安全に保存。
