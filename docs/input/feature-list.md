# 機能一覧

## 凡例
- 🔴 Must Have（MVP必須）
- 🟡 Should Have（MVP後優先）
- 🟢 Nice to Have（将来対応）

## 1. 旅程・計画機能
| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| 旅程検索 | エリア、子供の年齢、タグでプランを検索 | 🔴 | |
| 旅程詳細表示 | タイムライン形式でスポットと移動を表示 | 🔴 | 失敗/注意情報の強調 |
| プランコピー | 他者のプランを自分の下書きとして複製 | 🔴 | |
| プラン編集 | スポット追加・削除、時間変更 | 🔴 | ドラッグ＆ドロップUI |
| Google Map連携 | プランをGoogle Mapリスト/マイマップへ出力 | 🔴 | コア機能 |
| 旅のしおりPDF | 印刷用の可愛いしおり生成 | 🔴 | 差別化要因としてMVPに含める |

## 2. 投稿・振り返り機能
| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| 振り返り入力 | 実施したプランの確定、評価入力 | 🔴 | |
| 失敗情報タグ | 「トイレ汚い」「待ち時間長い」等の定型タグ | 🔴 | 選択式で簡単に入力 |
| 写真アップロード | 各スポットに対応する写真を登録 | 🔴 | ムービー生成用 |
| ムービー自動生成 | 旅程・コメント・写真を合成して動画化 | 🔴 | 投稿の主要インセンティブ |
| 公開設定 | プランを全体公開/非公開にする | 🔴 | |

## 3. コミュニティ・ゲーミフィケーション
| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| 役に立ったボタン | プランに対する感謝アクション | 🟡 | |
| ユーザーランク | 投稿数・評価に応じた称号（スタンプ） | 🟡 | パスポートのデザインに反映 |
