# ユーザーフロー

## 1. 計画・準備フェーズ
### 検索〜コピー
1. ユーザーはトップページ（空港の出発ロビー風）から「行き先」「子供の年齢」を入力。
2. 検索結果から、評価の高い旅程カード（航空券チケット風デザイン）を選ぶ。
3. 詳細画面でタイムラインと「⚠️注意ポイント」を確認。
4. 「このプランをCOPY」ボタンを押す。

### カスタマイズ〜出力
1. マイページ（パスポート風）にコピーされたプランが表示される。
2. ランチの場所を入れ替えたり、時間を調整する。
3. 「Google Mapへ書き出し」ボタンを押す。
4. スマホのGoogle Mapアプリが開き、スポットとお気に入りコメントが保存される。
5. （オプション）「旅のしおり」をPDFでダウンロード・印刷。

## 2. 旅行実行フェーズ
1. 現地ではGoogle Mapを開き、保存されたピンを頼りに移動。
2. ピンのメモ欄に「ここは〇歳児におすすめ」「トイレは2階」などの年齢別情報が入っているのを確認。

## 3. 振り返り・投稿フェーズ
### 投稿〜ムービー生成
1. 旅行後、サービスにアクセスし「振り返り」を開始。
2. 実施した旅程が表示される（変更があれば修正）。
3. 各スポットに「評価（★）」と「一言コメント（成功/失敗）」を入力。
4. スマホから主要な写真をアップロード。
5. 「投稿してムービーを作る」ボタンを押す。
6. 数秒〜数分で、時系列に写真とコメントが流れるショートムービーが完成。
7. 「ダウンロード」または「SNSシェア」。
8. プラン自体も公開され、他のユーザーが見られるようになる。

## 4. フィードバックループ
1. 公開したプランを別の誰かがコピーまたは参考にする。
2. 「役に立った！」ボタンが押される。
3. 投稿者に「あなたのプランが〇〇さんの旅行を助けました！」と通知（スタンプが増える）。
