# プロダクト提案

## 元のアイデア
子連れ旅行（特に小学生・幼児連れ）の計画を立てる際に、安全性・トイレ・食事などの細かい情報を効率的に収集・共有できるサービス。
Instagramやブログの情報分散を解決し、Google Map連携や「シェアしたくなる仕掛け」を取り入れたい。

## 提案一覧

| | 提案 | コンセプト | 特徴 | 向いている人 |
|---|------|------------|------|--------------|
| **A** | **旅程まるごとコピー型** | 「先輩家族の成功プラン」をそのまま真似る | 1日のスケジュール単位で共有。移動動線や休憩タイミングまで分かる。 | 計画の手間を極限まで減らしたい人。失敗したくない人。 |
| **B** | **ニッチ情報Wiki型** | 「ガイドブックに載っていないリアル」を集める | トイレ・食事・危険箇所などの細かい「地点情報」に特化。タグ検索が強力。 | 特定の場所の不安を解消したい人。細かい条件で探したい人。 |
| **C** | **マップリスト拡張型** | 「Google Map」を最強の子連れ地図にする | Google Mapのお気に入りリストをインポートして、コメントや評価を付与して共有。 | 普段Google Mapを使い倒している人。地図ベースで探したい人。 |
